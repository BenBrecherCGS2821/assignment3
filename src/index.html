Modern Web Development With Parcel Bundler, TypeScript, and LESS

By using a modern web development front-end stack, you can dramatically speed up your design and implementation. In this article, you will learn how to set up a NodeJS, how to use parcel, TypeScript and LESS.

TypeScript

TypeScript allows you to use an IDE with JavaScript and have static type information for your environment. This makes your code self-documenting and allows you to much quicker find and fix bugs, and make less bugs from the start. In order to use TypeScript, it first must be compiled by the TypeScript compiler.

LESS

LESS is a preprocessor for CSS that allows you to write styles far quicker and easier than plain CSS. LESS is very simple, and existing CSS is actually fully correct LESS. LESS Simply adds more features. You cannot actually use LESS directly in HTML. You have to run it through the LESS preprocessor first in order to get valid CSS which can be used in HTML.

Parcel Bundler

Parcel Bundler is where all the magic takes place. It will compile all your TypeScript, LESS and other assets into valid HTML that can be used on any browser. "Parcel has out of the box support for JS, CSS, HTML, file assets, and more - no plugins needed." (ParcelJS)
