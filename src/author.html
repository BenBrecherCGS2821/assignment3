Color Palette


Color Usage Reason
#343a40 Header/Footer Background Color This is the default color in the bootstrap dark theme. To keep it consistent, I'm using the same color in the footer. I chose the dark theme because it's commonly associated with developer's IDEs. Every experienced developer uses a dark theme becasue it's easier on the eyes, and is default for most IDEs now anyways.
#ffffff Header/Footer Text & Active Page Link & Content Background White simply has good contrast with dark colors. Note: Navigation links are the same color, but at half opacity.
#6b7886 Body Background I think it looks nice and makes the content easy to focus on. I'm trying something different. Surprisingly it doesnt look that bad, so I'm keeping it (for now). The blue hues complement my picture, in my opinion.
#000000 Text Content Standard black text to create high contrast color.
rgb(0, 123, 255) Hyperlink to External Page Standard blue link to external page. It's instantly recognizable. Why change it when it looks good already?
